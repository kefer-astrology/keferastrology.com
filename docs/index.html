Otevřený zdrojový kód

Zvolili jsme si cestu otevřeného softwaru, protože věříme, že tahle cesta pomůže jak nám (vše děláme vesvém volném čase) tak vám uživatelům a dalším nadšeným astrologům tvarovat tento program k obrazu svému. Célý zrdojový kód je otevřeně sdílen na platformě Github. Pokoušíme se ho co nejlépe dokumentovat, aby bylo co nejjedodušší měnit jednotlivé komponenty. Pro vlastní vývoj jsme vybrali programovací jazyk Rust (konkrétně podnož Tauri), který umožňuje spřažení s mocnými knihovnami programovacího jazyku Python a na druhé straně využít přesné polohování jednotlivých komponent pomocí Javascriptu (v našem případě se jedná o programovací jazyk Svelte Kit).
